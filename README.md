# Music-Player
*Музыкальный плейер с выбранными композициями*

## [Посмотреть и потыкать]()

В функционал входят:
- ползунок громкости
- ползунок трека
- базовые кнопки (остановка, вперед, назад)
- визуализация звуковой дорожки
- кнопка для включения случайной композиции
- повтор трека
- крутящаяся обложка
